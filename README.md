# 行星轨道数值模拟

## 项目简介

本项目通过数值求解牛顿万有引力定律下的二体问题，模拟行星绕太阳的运动轨迹。使用 Python 的 `scipy.integrate.solve_ivp` 进行高精度数值积分，直观展示不同类型的行星轨道。

---

## 物理模型

### 基本假设
- 太阳固定不动（M☀ ≫ m_planet）
- 二维平面运动
- 仅考虑引力作用

### 运动方程

根据牛顿第二定律和万有引力定律：

$$m\frac{d^2\vec{r}}{dt^2} = -\frac{GM_\odot m}{r^2}\hat{r}$$

消去行星质量 $m$：

$$\frac{d^2\vec{r}}{dt^2} = -\frac{GM_\odot}{r^3}\vec{r}$$

化为一阶微分方程组，定义状态向量 $\vec{y} = (x,\ y,\ v_x,\ v_y)^T$：

$$\begin{cases}
\dfrac{dx}{dt} = v_x \\[10pt]
\dfrac{dy}{dt} = v_y \\[10pt]
\dfrac{dv_x}{dt} = -\dfrac{GM_\odot}{r^3}x \\[10pt]
\dfrac{dv_y}{dt} = -\dfrac{GM_\odot}{r^3}y
\end{cases}$$

其中 $r = \sqrt{x^2 + y^2}$。

### 无量纲化

采用天文单位制（AU、年、太阳质量），由开普勒第三定律 $T^2 = \frac{4\pi^2}{GM_\odot}a^3$，代入 $T=1,\ a=1$：

$$GM_\odot = 4\pi^2 \approx 39.478$$

| 物理量 | 单位 | 数值 |
|--------|------|------|
| 长度 | AU（天文单位） | 1 AU = 1.496×10¹¹ m |
| 时间 | 年（yr） | 1 yr = 3.156×10⁷ s |
| 质量 | M☀（太阳质量） | 1 M☀ = 1.989×10³⁰ kg |

### 数值方法
自适应步长 Runge-Kutta 法（`solve_ivp`），误差容限：
- 相对误差：`rtol = 1e-9`
- 绝对误差：`atol = 1e-12`

## 主要功能

### 1. 地球轨道模拟
以地球参数为初始条件（日地距离 1 AU，轨道速度 2π AU/yr），模拟 2 年的轨道运动。

**模拟结果：**
- 轨道平均距离：1.000000 AU
- 距离标准差：≈ 0（完美圆形）
- 平均速率：6.2832 AU/yr

### 2. 多维度可视化
生成四合一分析图表：
- **轨道形状图** — 行星绕日运动轨迹
- **距离变化图** — 日地距离随时间变化
- **速度分量图** — vx、vy 随时间振荡
- **速率变化图** — 验证匀速圆周运动

### 3. 守恒量验证
- 总机械能相对误差 < 0.0001%
- 角动量守恒精度极高
- 数值求解可靠性得到保证

### 4. 轨道类型对比
模拟 7 种不同初速度，展示圆锥曲线族：

| 初速度 (AU/yr) | 轨道类型 |
|----------------|----------|
| 4.0 ~ 6.0 | 椭圆（太阳在远焦点） |
| 2π ≈ 6.283 | 正圆 |
| 7.0 ~ 8.5 | 椭圆（太阳在近焦点） |
| 8.886 | 抛物线（逃逸临界） |
| > 8.886 | 双曲线（行星逃逸） |

其中逃逸速度 $v_{\text{escape}} = \sqrt{2}\ v_{\text{circular}} = \sqrt{2} \times 2\pi \approx 8.886$ AU/yr。

## 开普勒定律验证情况

| 定律 | 验证状态 | 说明 |
|------|----------|------|
| 第一定律（轨道定律） | ✅ 已验证 | 展示了椭圆轨道，太阳位于焦点 |
| 第二定律（面积定律） | ⚠️ 间接支持 | 角动量守恒⇒面积速度恒定，但未直接计算 |
| 第三定律（周期定律） | ❌ 未验证 | 仅模拟单一行星，未比较 T² ∝ a³ |
